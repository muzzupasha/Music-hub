# 🎵 Music Hub

Welcome to **Music Hub** – a state-of-the-art platform dedicated to discovering, learning, and mastering music. Built with cutting-edge web technologies, Music Hub offers an immersive, interactive learning experience featuring dynamic micro-animations, rich aesthetics, and a sleek, premium dark-mode design.

🚀 **Live Demo:** [https://music-hub-g.netlify.app/](https://music-hub-g.netlify.app/)

## ✨ Features

- **Premium UI/UX:** Stunning visual aesthetics powered by Aceternity UI and Framer Motion, delivering glassmorphism effects, dynamic moving borders, tooltips, and particle animations.
- **Extensive Course Library:** A comprehensive catalog of expertly crafted music courses ranging from *Basic Music Theory* to *Advanced Composition* and *Music Production*.
- **Interactive Elements:** Features highly engaging components including 3D sticky scrolling, glare cards, infinite moving review cards, and complex parallax hero sections.
- **Ultra Responsive:** Fully optimized architecture seamlessly adapting to all devices—desktops, tablets, and smartphones.

## 🛠️ Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (v15+ App Router, Server Components)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/) (v11), [@tsparticles](https://particles.js.org/)
- **UI Patterns:** Custom-built component library utilizing Aceternity UI design systems.
- **Deployment:** [Netlify](https://www.netlify.com/)

## 🚀 Getting Started

Follow these instructions to run the project locally on your machine:

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) (v18 or higher) installed on your system.

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/muzzupasha/Music-hub.git
   cd Music-hub
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Access the application**
   Open your browser and navigate to [http://localhost:3000](http://localhost:3000).

## 📦 Build & Deployment

This project strictly adheres to Next.js production standards and is optimized for zero-config deployments on Netlify and Vercel.

To create an optimized production build locally:
```bash
npm run build
npm start
```
